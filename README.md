# ByteDojo

A CLI tool for fetching, solving, and tracking programming problems from platforms like LeetCode. Master coding through structured repetition and spaced review.

## Installation

```bash
# Clone and install
git clone https://github.com/stephen-os/bytedojo.git bytedojo
cd bytedojo
pip install -e .
```

## Options

```bash
dojo --help          # Show help
dojo --version       # Show version
dojo --author        # Show author info
dojo --desc          # Show full description
dojo --debug [cmd]   # Enable debug mode
```
